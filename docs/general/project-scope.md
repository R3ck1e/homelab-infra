# Project Scope

homelab-infra is a learning-oriented infrastructure project
designed to simulate production-grade DevOps practices
on limited hardware resources.

---

## Objectives

- Practice Infrastructure as Code (Terraform).
- Deploy and manage Kubernetes clusters.
- Implement GitOps workflows (ArgoCD / ApplicationSets).
- Implement observability stack.
- Simulate disaster recovery strategies.
- Introduce production-like governance and documentation.

---

## Constraints

- Single engineer.
- Limited hardware (local hypervisor).
- No external SaaS dependency for core infrastructure.
- Educational focus with production mindset.

---

## Architectural Philosophy

- Clear responsibility boundaries.
- Deterministic infrastructure (static addressing where needed).
- Reproducible environments.
- Minimal but structured governance.
- Progressive complexity (layer-by-layer expansion).
